# BoomPets - Smart Nutrition for Every Pet

AI-powered pet nutrition app that provides personalized meal plans, portion calculations, and diet management for dogs, cats, birds, and fish.

## Features

- Pet profile onboarding with photo upload
- AI-powered meal recommendations
- Portion calculator based on pet weight
- Diet management and tracking
- Food logging with calorie tracking
- Health & allergy management
- Environmental context awareness

## Tech Stack

- React Native with Expo
- React Navigation
- Context API for state management

## Getting Started

```bash
npm install
npm start
```
